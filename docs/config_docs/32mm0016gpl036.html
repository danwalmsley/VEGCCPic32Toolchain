Configuration-Bit Settings for PIC32MM0016GPL036

(Data Sheet # DS-00000000)

Usage:
#pragma config SETTING = VALUE

Alternate Comparator1 Pins Location Enable bit (ALTCMP)
ALTCMP = ON Alternate Comparator1 pins are used
ALTCMP = OFF Primary Comparator1 pins are used

Secondary Oscillator High Power Enable bit (SOSCHP)
SOSCHP = ON SOSC oprerates in high power mode
SOSCHP = OFF SOSC oprerates in normal power mode.

Alternate I2C1 Pins Location Enable bit (ALTI2C)
ALTI2C = ON Alternate I2C1 pins are used
ALTI2C = OFF Primary I2C1 pins are used

User ID bits (USERID)

JTAG Enable bit (JTAGEN)
JTAGEN = ON JTAG is enabled
JTAGEN = OFF JTAG is disabled

ICE/ICD Communication Channel Selection bits (ICS)
ICS = PGx1 Communicate on PGEC1/PGED1
ICS = PGx2 Communicate on PGEC2/PGED2
ICS = PGx3 Communicate on PGEC3/PGED3
ICS = NONE Not connected

Brown-out Reset Enable bits (BOREN)
BOREN = BOR3 Brown-out Reset enabled in hardware; SBOREN bit disabled
BOREN = BOR2 Brown-out Reset enabled only while device active and disabled in Sleep; SBOREN bit disabled
BOREN = BOR1 Brown-out Reset controlled with the SBOREN bit setting
BOREN = BOR0 Brown-out Reset disabled in hardware; SBOREN bit disabled

Retention Voltage Regulator Enable bit (RETVR)
RETVR = ON Retention regulator is enabled and controlled by RETEN bit during sleep
RETVR = OFF Retention regulator is disabled

Downside Voltage Protection Enable bit (DNVPEN)
DNVPEN = ON Downside protection is enabled
DNVPEN = OFF Downside protection is disabled

SWDTPS = PS1 1:1
SWDTPS = PS2 1:2

SWDTPS = PS8 1:8
SWDTPS = PS16 1:16

SWDTPS = PS131072 1:131072
SWDTPS = PS262144 1:262144
SWDTPS = PS524288 1:524288
SWDTPS = PS1048576 1:1048576

Watchdog Timer Window Size bits (WDTWIN)
WDTWIN = PS75_0 Watchdog timer window size is 75%
WDTWIN = PS50_0 Watchdog timer window size is 50%
WDTWIN = PS37_5 Watchdog timer window size is 37.5%
WDTWIN = PS25_0 Watchdog timer window size is 25%

Windowed Watchdog Timer Enable bit (WINEN)
WINEN = ON Watchdog timer is in window mode
WINEN = OFF Watchdog timer is in non-window mode

Run Mode Watchdog Timer Clock Source Selection bits (RCLKSEL)
RCLKSEL = SYSCLK Clock source is system clock
RCLKSEL = FRC Clock source is FRC oscillator
RCLKSEL = LPRC Clock source is LPRC (same as for sleep mode)

Watchdog Timer Enable bit (FWDTEN)
FWDTEN = ON WDT is enabled
FWDTEN = OFF WDT is disabled

Oscillator Selection bits (FNOSC)
FNOSC = FRCDIV Fast RC oscillator (FRC) with divide-by-N
FNOSC = PLL Primary or FRC oscillator with PLL
FNOSC = PRI Primary oscillator (XT, HS, EC)
FNOSC = DCO Digitally controlled oscillator (DCO)
FNOSC = SOSC Low power secondary oscillator (SOSC)
FNOSC = LPRC Low power RC oscillator (LPRC)

System PLL Input Clock Selection bit (PLLSRC)
PLLSRC = PRI Primary oscillator is selected as PLL reference input on device reset
PLLSRC = FRC FRC oscillator is selected as PLL reference input on device reset

Secondary Oscillator Low Power Mode Enable bit (LPOSCEN)
LPOSCEN = ON Secondary oscillator (SOSC) low power mode is enabled
LPOSCEN = OFF Secondary oscillator (SOSC) low power mode is disabled

Two Speed Startup Enable bit (IESO)
IESO = ON Two speed startup is enabled
IESO = OFF Two speed startup is disabled

Primary Oscillator Selection bit (POSCMOD)
POSCMOD = EC External clock mode is selected
POSCMOD = XT XT oscillator mode is selected
POSCMOD = HS HS oscillator mode is selected
POSCMOD = OFF Primary oscillator is disabled

System Clock on CLKO Pin Enable bit (OSCIOFNC)
OSCIOFNC = ON System clock is connected to CLKO/OSC2 pin
OSCIOFNC = OFF OSCO pin operates as a normal I/O

Secondary Oscillator External Clock Enable bit (SOSCSEL)
SOSCSEL = OFF Crystal is used
SOSCSEL = ON External clock connected to SOSCI pin is used

Clock Switching and Fail-Safe Clock Monitor Enable bits (FCKSM)
FCKSM = CSDCMD Clock switching is disabled; Fail-safe clock monitor is disabled
FCKSM = CSECMD Clock switching is enabled; Fail-safe clock monitor is disabled
FCKSM = CSDCME Clock switching is disabled; Fail-safe clock monitor is enabled
FCKSM = CSECME Clock switching is enabled; Fail-safe clock monitor is enabled

Code Protection Enable bit (CP)
CP = ON Code protection is enabled
CP = OFF Code protection is disabled
