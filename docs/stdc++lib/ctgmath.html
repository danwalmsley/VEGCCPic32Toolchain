<ctgmath>

[added with C99]

Include the standard header <ctgmath> to define the macros traditionally defined in the Standard C library header <tgmath.h>. Including this header also ensures that the names declared with external linkage in the Standard C library header are declared in the std namespace. In this implementation, the names may or may not also be declared in the global namespace, depending on the specific translation environment.
