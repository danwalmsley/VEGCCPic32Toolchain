The XC License Manager (XCLM) in MPLAB X IDE

The Microchip Technology XC License Manager (XCLM) is a cross-platform license manager for MPLAB® XC C compilers.

Some XCLM functions are supported in MPLAB X IDE.

Using the XC License Manager (XCLM) to Roam

The XC License Manager (XCLM) supports a network license server/client setup. You install a network license server and then check out client licenses. See the installation documenation - "Installing and Licensing MPLAB© XC C Compilers" (DS52059) - for details on how to install a network license server and connect clients.

As a Network client, you can checkout a network license that is active for up to 30 days by using Roam Out. Reasons to roam out may be:

  • You want to hold on to a license for several days.
  • You have a laptop and will not have a consistent connection to a network.

The license server will check out and give a license to you. The license server now has one less license of product that it can give out. The license will stay checked out until you chose to Roam In or 30 days have passed.

In the MPLAB X IDE, you can easily roam your XC network license in or out.

  1. Select Tools>Licenses>Roam Network License to open the Network License Management dialog (see image below).
  2. Click on the compiler that you wish to roam in or out.
  3. Click on either the "Roam in" or "Roam out" green arrows.
    Note: When you roam in, there is a 60-minute delay before someone else can use your license.
  4. Click Close when you are done.

You can use roam in and roam out on the command line as well. For details, see Command-Line Options Reference.

Using the XC License Manager (XCLM) to a Active Workstation License

If you have a free or evalution version of the XCLM, you can activate it from MPLAB X IDE by doing the following:

  1. Select Tools>Licenses>Activate Workstation License to open the Activate Workstation License dialog (see image below).
  2. Type in your license.
  3. Click Activate to activate the license.
