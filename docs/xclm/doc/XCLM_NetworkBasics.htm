XCLM Network Setup

The Microchip Technology XC License Manager (XCLM) is a cross-platform license manager for MPLAB® XC C compilers.

XCLM is a modification of the Reprise License Manager™ (RLM) for managing its language tool licensing. For more on the RLM, see the supplementary user’s guide, XCLM_Advanced.html which can be found in the docs folder of the compiler installation.

Network installation occurs in a multi-step process: a one time setup of the network server, and a quick configuration of every compiler that will connect to the network server.
