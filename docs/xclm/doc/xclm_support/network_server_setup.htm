How to start the XCLM server

To start the network server, there are 2 necessary components: server binaries and network licenses

Creating licenses

Note: If all necessary licenses were created during installation, skip this step and go straight to 'Starting the server'

Point a command prompt or terminal to the binary folder located under the Microchip product tree in the path Microchip/xc_license_server/v#.##/bin/

To create a license, execute the command:
> xclm -actkey <activation-key>

This option pings the server with the activation-key specified to license your XC compiler. For full documentation on this option visit the command line reference.


Starting the server

Server binaries are located under the Microchip product tree in the path Microchip/xc_license_server/v#.##/bin/

The server binary is named rlm.exe

Finding the license folder

License files are located in different folders depending on the operating system

The license folder can be found by running:
> xclm -licensepath

Starting the license server

To start the license server execute the command: rlm -c <license folder path>

If started correctly, you should see output like this:
