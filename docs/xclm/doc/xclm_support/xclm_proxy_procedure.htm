Activating a License From Behind a Proxy

To use a proxy server, there are 2 environment variables which must be set:

HTTP_PROXY - the hostname and port of the proxy server, in the format hostname:port.

If your proxy server uses authentication, you can use the HTTP_PROXY_CREDENTIALS environment variable to pass the credentials to the proxy server:
HTTP_PROXY_CREDENTIALS - the username and password to authenticate you to the proxy server, in the format user:password.


For example, if your proxy server is on port "8080" on host "proxy_host" and
if your username is "john" and password is "johns_password", the procedure on Windows would look like:

> set HTTP_PROXY=proxy_host:8080
> set HTTP_PROXY_CREDENTIALS=john:johns_password
> xclm -actkey XXXX-XXXX-XXXX-XXXX
Note that xclm activation supports only the BASIC authentication type.
