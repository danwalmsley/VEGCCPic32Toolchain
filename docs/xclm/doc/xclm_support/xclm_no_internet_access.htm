Creating a License for a Computer without Internet Access

1. On the PC which does not have internet connectivity (also referred to as the offline computer), run the following command:
> xclm -hostinfo
The command will print out information about the offline computer. This information should be used in the next step.

2. Run the following commands on a PC which has internet connectivity.

To activate a license for a single, node-locked computer (most likely scenario), run the command:
> xclm -actkey <activation-key> hostid=<mac>
To activate a license for a network server, run the command:
> xclm -actkey <activation-key> hostid=<mac> hostname=<host>

Example:
> xclm -actkey 1234-1234-1234-1234 hostid=0123456789ab

<activation-key> is the key that was given to you at the time of the compiler purchase.
<mac> must be the valid Media Access Control (mac) address of the offline computer.
<host> must be the valid username of the offline computer.

3. The previous command will create a license file (e.g. microchip1.lic) and display where the license was created.

4. Copy this .lic file to the license folder on the computer without internet access. The license folder can be found by running:
> xclm -licensepath

5. Install the XC compiler in FREE mode on the computer which does not have internet connectivity.

6. The XC compiler will now operate with licensed capabilities.
