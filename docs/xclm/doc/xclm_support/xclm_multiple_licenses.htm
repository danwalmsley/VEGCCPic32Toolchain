Activating Multiple Licenses

If you wish to activate several licenses, it is possible to activate all of them easily through the command line with the use of a scripting language. This is easier than activating licenses one at a time through the installer.

Depending on the script structure, the activation keys could be stored in a file where the script can look for keys, or provided through the command line as an argument to the program. Once the program has a list of all activation keys, the script should execute xclm{.exe} -actkey for every activation key in the list.

This is an example of a script that will activate multiple keys:

File: activate_multiple_keys.sh

#!/bin/sh

if [ $# -eq 0 ]; then
  echo “Please input at least one activation key into this program”;
  exit;
fi
if [ ! -x xclm ]; then
  echo “xclm binary not found”;
  echo “Please install latest compiler to get the xclm binary;
  exit;
fi

for input_arg in "$@"; do
  xclm -actkey $input_arg;
done

This script can run natively on Linux and Mac OS systems. It can also be run on Windows OS systems by installing Cygwin. The program should be executed like this

./activate_multiple_keys.sh ...
