The XC License Manager (XCLM) on the Command Line

The Microchip Technology XC License Manager (XCLM) is a cross-platform license manager for MPLAB® XC C compilers.

The XCLM is a modification of the Reprise License Manager™ (RLM) for managing its language tool licensing. For more on the RLM, see the supplementary user’s guide:
<default OS application folder>/microchip/xclm/doc/RLM_Enduser.html

The XCLM is installed when an MPLAB XC C compiler is installed using the GUI installer, instructions for which are located on the MPLAB XC C Compilers webpage. Once installed, you may use the XCLM on the command line to complete additional tasks. These are described below.

  • Displaying Current Status of Licenses
  • Creating a License for a Computer with Internet Access
  • Creating a License for a Computer without Internet Access
  • Activating Multiple Licenses
  • Activating a License From Behind a Proxy
  • Creating a Demo License
  • How to start the XCLM server
  • Configuring a Client to Point to a Network Server
